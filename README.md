RETAIL SALES CLI APPLICATION


A Simple Cli Application Where A Retail Company Can Enter Sales Data For Their Branches Across 12 Months

Compile Method: 

gcc main.c logic.c -o name_of_program


Run Mehtod:

Windows: 

CMD/Powershell: main


Bash Shell/Linux Terminal: ./main

Features: 

1.Input Sales

2.Print Sales

3.Determine Month With Peak Sales

4.Determine Branch Percentages

5.Sort Each Branches By Month / And Each Months By Branch

6.Data Persistance
